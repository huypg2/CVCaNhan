Remote Command Line

PYTHON • SUBPROCESS • SOCKET

1. Giới thiệu

Mô phỏng chức năng của Telnet/SSH. Client gửi câu lệnh (ví dụ: `dir`, `ipconfig`), Server thực thi lệnh đó trên máy của mình và trả kết quả về cho Client xem.

2. Code Demo (Server thực thi lệnh)


import socket
import subprocess

server = socket.socket()
server.bind(('0.0.0.0', 9999))
server.listen(1)
client, addr = server.accept()

while True:
    command = client.recv(1024).decode()
    if command.lower() == 'exit':
        break
    
    output = subprocess.getoutput(command)
    client.send(output.encode())

client.close()
